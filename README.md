## Overview

This project analyzes startup financial data to identify key factors affecting profitability. Using Power BI dashboards and regression analysis, the project uncovers relationships between R&D spend, marketing, administration costs, and profit outcomes.

---

## Objectives

- Identify the key drivers of startup profit
- Analyze the impact of R&D, Marketing, and Administration spending
- Build a regression-based predictive model for profit estimation
- Deliver interactive dashboards for business decision-making

---

## Tools & Technologies

| Tool | Purpose |
|------|---------|
| Power BI | Dashboard creation & interactive visualizations |
| Excel | Data preprocessing & statistical summary |
| SQL | Data querying and preparation |

---

## Dataset

Startup financial dataset containing:

| Column | Description |
|--------|-------------|
| R&D Spend | Investment in research and development |
| Marketing Spend | Budget allocated to marketing |
| Administration | Overhead and administrative costs |
| State | Geographic region of operation |
| Profit | Target variable — net profit |

---

## Key Insights

- **R&D Spend** has the strongest positive correlation with profit
- **Marketing Spend** shows moderate influence on profitability
- **Administration costs** have minimal impact on profit
- Regression model achieved **R² = 0.95**, indicating high predictive accuracy

---

## Dashboard Features

- Interactive slicers for filtering by state and spending categories
- What-if analysis for profit forecasting
- KPI cards displaying total revenue, cost, and profit metrics
- Trend charts for R&D vs Profit correlation
